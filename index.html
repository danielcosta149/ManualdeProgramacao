    • GIT - GITHUB

    Olá, seja bem vindo à seção GIT - GITHUB. Neste espaço você encontrará informações e dicas sobre como utilizar o Git e o GitHub para versionamento de código, colaboração em projetos e controle de versões. Explore os tópicos ao lado para começar!

    Primeiros Passos:

    Antes de você começar a conhecer os "códigos" do Git, você precisa saber a parte da criação de uma conta no GitHub e como deve ser feita a instalação.



    1. Criando uma conta no GitHub:

    • Acesse o site github.com e clique em "Sign up".
    • Preencha seus dados (e-mail, senha, nome de usuário) e siga as instruções para confirmar sua conta.

    2. Instalando o Git:

    • Windows: Baixe o instalador em git-scm.com e siga o assistente de instalação.
    • Mac: Use o Homebrew com o comando brew install git ou baixe em git-scm.com.
    • Linux: Use o gerenciador de pacotes, por exemplo: sudo apt-get install git (Debian/Ubuntu).

    Após instalar, abra o terminal e digite git --version para verificar se a instalação foi bem-sucedida.
